# img2pdf

#### 介绍
将图片转为pdf

#### 使用说明
0.  需要提前安装pillow或PIL
1.  将图片文件放在同一文件夹下
2.  python3 img2pdf.py [input_path] [output_path]
3.  参数：
    input_path（可选）:
        输入图片文件夹的地址，没有该参数则在当前目录下寻找.
    output_path（可选）:
        输出保存的地址，没有该参数则保存在./pdf下.
